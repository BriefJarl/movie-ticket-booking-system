# 🎬 Movie Ticket Booking System



A full-stack backend project built using Spring Boot, Kafka, and MySQL.



## 🚀 Features

- 🎟️ Book movie tickets

- 💺 Real-time seat availability

- ⚡ Kafka event-driven architecture

- 📩 Email notification on booking

- 👤 User-based booking history



## 🛠️ Tech Stack

- Java + Spring Boot

- Apache Kafka

- MySQL

- Spring Data JPA

- REST APIs



## 🔥 Flow

1. User books ticket

2. TicketService processes booking

3. Kafka event is produced

4. Consumer listens to event

5. Email is sent automatically



## 📸 Screenshots

- Booking API response

- Kafka console logs

- Email confirmation



## ▶️ How to Run

1. Start Kafka + Zookeeper

2. Run Spring Boot app

3. Hit API: `/kafka/book?user=...\&movie=...\&seat=...`
